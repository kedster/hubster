About Tool Hub

Tool Hub is an open-source collection of web-based utilities designed to streamline your workflow and boost productivity.

Features

  • Dynamic tool discovery and loading
  • Responsive design that works on all devices
  • Fast, client-side tool execution
  • Easy to extend with new tools
  • Modern, intuitive interface

Adding New Tools

To add a new tool to the hub:

  1. Create a new folder in the /tools directory
  2. Add an index.html file with your tool
  3. Create a config.json file with tool metadata
  4. Run the build script: node scripts/build.js build
  5. The tool will be automatically discovered and added to the hub

Development

To develop and test locally:

  1. Build the project: node scripts/build.js build
  2. Start development server: node scripts/build.js serve
  3. Open http://localhost:3000 in your browser
